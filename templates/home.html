Selamat Datang di HR Attrition Analytics

Tentang Aplikasi

Aplikasi ini dirancang untuk menganalisis dan memprediksi faktor-faktor yang mempengaruhi attrition (turnover) karyawan. Menggunakan Machine Learning dan visualisasi data interaktif, kami membantu Anda memahami pola-pola yang berkontribusi terhadap keputusan karyawan meninggalkan perusahaan.

Fitur Utama

📊 Dashboard Analitik

Visualisasi komprehensif tentang tren attrition dan insight bisnis menggunakan Google Looker Studio.

Lihat Dashboard →

🔮 Prediksi Attrition

Prediksi attrition untuk karyawan individual berdasarkan karakteristik dan faktor-faktor kerja mereka.

Buat Prediksi →

🤖 Model Management

Tracking dan monitoring model ML dengan MLflow dan DagsHub untuk versioning dan kolaborasi.

📈 Real-time Analytics

Analisis real-time dengan data terintegrasi dan dashboard yang selalu ter-update.

Bagaimana Cara Menggunakan?

1

Jelajahi Dashboard

Lihat analisis mendalam tentang faktor-faktor attrition melalui dashboard interaktif

2

Input Data Karyawan

Masukkan informasi karyawan di form prediksi untuk analisis individual

3

Dapatkan Insights

Terima prediksi dan rekomendasi berdasarkan model machine learning
